> Actions required from the store owner in **Shopify Admin** to make the Header and Category Pills sections fully functional.

## 🧭 Header / Navigation

### 1. Create Main Navigation Menu
- Go to: **Shopify Admin → Online Store → Navigation**
- Create a new menu with the handle: `main-menu`
- Add the following 5 links (in order):

| Label | Destination |
|---|---|
| Featured | `/collections/featured` (or any collection) |
| Sneakers | `/collections/sneakers` |
| Streetwear | `/collections/streetwear` |
| Accessories | `/collections/accessories` |
| Collectibles | `/collections/collectibles` |

> **Note:** If collections don't exist yet, use `/collections/all` as a temporary destination for each — swap them once collections are created.

## 🟪 Category Pills

### 2. Assign Thumbnail Images to Each Pill
- Go to: **Shopify Admin → Online Store → Customize**
- Open the **Homepage**
- Click the **Category Pills** section
- For each pill, click it and upload a thumbnail image (square crop recommended, minimum 200×200px):

| Pill | Suggested image |
|---|---|
| Trending Now | A trending product shot or lifestyle image |
| Pokemon Collectibles | Pokemon card/figure shoot |
| Chrome Hearts | Chrome Hearts jewelry or tee |
| Loewe x On | Loewe x On sneaker shot |

### 3. Set pill links to correct collections
- In the same Category Pills section in the Customizer, update each pill's **Link** field to the correct collection URL once the collections are live.
